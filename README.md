# 峰运通数据管理系统

面向峰运通内部使用的 Windows 桌面数据管理工具，将考勤填报、工时对账、到料明细、透视表制作等日常工作集成到一个程序中，并内置数据库与一批常用小工具，开箱即用，支持一键在线更新。

## 功能

**核心业务**

- **考勤填报**：批量处理考勤数据并生成考勤填报表。
- **工时对账**：核对工时记录，快速定位差异。
- **到料明细**：整理、汇总到料明细数据。
- **透视表制作**：按所选维度一键生成 Excel 透视表。

**数据与工具箱**

- **数据库**：把各处表格拖进来自动识别归档，随取随用。
- **批量重命名**：按规则批量改名，实时预览、可撤销。
- **金额大写**：数字金额一键转规范中文大写人民币，开票、报销、合同可直接用。
- **文本工具箱**：文本去重、排序、去空行、去空格、大小写转换、加行号、统计，以及提取邮箱 / 手机号 / 网址。
- **PDF 工具箱**：合并多个 PDF、按页拆分、提取或删除指定页，纯本地处理不上传。
- **Excel 工具箱**：多工作簿合并、按工作表拆分、xls/xlsx/CSV 格式互转、多张同结构表纵向合并；拆分与合并会**保留字体、填充、边框、列宽、合并单元格等原始格式**，合并时可选保留公式。

## 下载与安装

1. 打开[最新发布页](https://github.com/KuroNeko-night/fyt-data-mgmt/releases/latest)
2. 在 **Assets** 中下载 `FYTDataMgmt_Setup_x.x.x.exe`
3. 双击运行，按向导完成安装；安装后会在桌面和开始菜单生成快捷方式
4. 若过程中弹出系统权限（UAC）提示，选择「是」

## 在线更新

程序内置在线更新。进入「关于 / 更新」页，点击 **检查更新**：

- 已是最新版本时会直接提示。
- 发现新版本时，按钮变为「下载并安装」，点击后自动下载安装包并显示进度，下载完成后启动安装向导，旧程序会自动关闭。按向导装完重新打开即可。

## 数据存放位置

程序的配置、导出结果、数据库与运行日志均保存在当前用户的文档目录下：

```
文档\峰运通数据管理系统\
```

该目录独立于程序安装目录，**卸载或更新程序都不会删除你的数据**。

## 系统要求

- 操作系统：Windows 7 SP1 / 8 / 10 / 11（64 位）
- 磁盘空间：约 250 MB 可用空间
- 无需联网即可使用；检查更新时才需要网络
