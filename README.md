# 峰运通数据管理系统

面向峰运通内部使用的数据管理工作台，以 Tauri 桌面端和局域网 Web 端承载考勤填报、工时对账、到料明细、销售透视等业务，并复用同一套 Python 核心。

> Tauri 桌面应用 + 局域网 Web 服务 · 支持 Windows 10/11（64 位）· 安装版无需预装 Python。

## 功能

**核心业务**

- **考勤填报**：批量处理考勤数据并生成考勤填报表。
- **工时对账**：核对工时记录，快速定位差异。
- **到料明细**：整理、汇总到料明细数据。
- **透视表制作**：按所选维度一键生成 Excel 透视表。

**数据与工具箱**

- **数据库**：把各处表格拖进来自动识别归档，随取随用。
- **任务中心**：自动记录后台处理任务的状态、耗时、结果位置与失败原因。
- **批量重命名**：按规则批量改名，实时预览、可撤销。
- **金额大写**：数字金额一键转规范中文大写人民币，开票、报销、合同可直接用。
- **文本工具箱**：文本去重、排序、去空行、去空格、大小写转换、加行号、统计，以及提取邮箱 / 手机号 / 网址。
- **PDF 工具箱**：合并多个 PDF、按页拆分、提取或删除指定页，纯本地处理不上传。
- **Excel 工具箱**：多工作簿合并、按工作表拆分、xls/xlsx/CSV 格式互转、多张同结构表纵向合并；拆分与合并会**保留字体、填充、边框、列宽、合并单元格等原始格式**，合并时可选保留公式。

**使用体验**

- **选择即预检**：选好文件立刻在页内提示表头识别情况；认不出表头时按数据形态兜底猜列并提醒核对，生成前就能发现列错位。
- **文件预览**：右侧面板可预览所选表格前若干行、切换子表；复核 / 结果 / 预览多个分区可同屏共存、各自折叠或收起。
- **不打断**：处理结果、完成与错误一律页内通知条提示；删除、重命名等操作用页内确认条二次确认，不再弹窗打断。

## 下载与安装

1. 打开[最新发布页](https://github.com/KuroNeko-night/fyt-data-mgmt/releases/latest)
2. 在 **Assets** 中下载 `FYTDataMgmt_Setup_x.x.x.exe`
3. 双击运行，按向导完成安装；安装后会在桌面和开始菜单生成快捷方式
4. 若过程中弹出系统权限（UAC）提示，选择「是」

## 在线更新

程序内置在线更新。进入「关于 / 更新」页，点击 **检查更新**：

- 已是最新版本时会直接提示。
- 发现新版本时，按钮变为「下载并安装」，点击后自动下载安装包并显示进度，下载完成后启动安装向导，旧程序会自动关闭。按向导装完重新打开即可。

## 数据存放位置

程序的配置、导出结果、数据库与运行日志均保存在当前用户的文档目录下：

```
文档\峰运通数据管理系统\
```

该目录独立于程序安装目录，**卸载或更新程序都不会删除你的数据**。

## 系统要求

- 操作系统：Windows 10 / 11（64 位）
- 磁盘空间：约 250 MB 可用空间
- 无需联网即可使用；检查更新时才需要网络

## 开发与发布

技术栈：React 19 · Vite 8 · Tauri 2 · Rust stable MSVC · Python 3.13 业务核心 · SQLite · openpyxl · xlrd(读老 .xls) · pypdf。PySide6 仅用于 Web 服务启停控制台，不再提供旧 Qt 业务入口。

**准备环境**

```powershell
.\setup-modern.ps1
```

**本地运行 Tauri 桌面端**

```powershell
cd tauri-app
npm run tauri dev -- --no-watch
```

**前端回归检查**

```powershell
cd tauri-app
npm run qa:visual
npm run qa:functional
```

两项检查分别覆盖 20 个页面的滚动、引导、主题/动效和响应式布局，以及输入切换、异步设置载入、重复任务和旧结果失效等桥接状态场景。

**打包 Tauri 安装版**

```powershell
cd tauri-app
npm run tauri:build
```

脚本会自动构建不含 Qt 的 `FYTCoreBridge.exe`，再生成简体中文 MSI 与 NSIS 安装包，产物在 `tauri-app/src-tauri/target/release/bundle/`。Cargo 依赖使用 `rsproxy.cn`，安装器工具默认使用 `ghfast.top` 镜像。



项目新增了独立的浏览器入口，支持在局域网内登录、注册审核、管理员账号管理，以及通过上传文件执行现有 Python 核心业务并下载结果。详见 [`web-app/README.md`](web-app/README.md)。

```powershell
npm --prefix web-app install
npm --prefix web-app run build
.\run-web.ps1
```

需要图形化启停时运行 `.\run-web-gui.ps1`，可在窗口中设置监听端口、启动或关闭 Web 服务并复制访问地址；控制台和其启动的服务均使用无命令窗口模式。直接运行 `.\run-web.ps1` 时服务默认在后台隐藏启动，排查问题可使用 `.\run-web.ps1 -Foreground` 查看原始控制台日志。

默认地址为 `http://本机IP:8787/`，默认管理员为 `admin / admin123456`，上线前请使用 `-AdminPassword` 修改。
